# 重力翻转迷宫

一个纯前端的轻量网页版小游戏。玩家不能直接控制移动方向，只能翻转重力，让小球滚动、坠落并抵达终点。

## 运行方式

直接打开 `index.html` 即可运行。

如果要部署到 GitHub Pages：

1. 把整个 `重力翻转迷宫` 文件夹上传到仓库。
2. 确保 Pages 指向这个静态站点目录。
3. 打开生成的页面链接即可游玩。

## 当前版本内容

- 原生 `HTML + CSS + JavaScript`
- `Canvas` 渲染
- 键盘与触屏双端重力控制
- 12 个关卡，分成 3 个难度梯度
- 失败快速重开
- 通关弹层与粒子反馈
